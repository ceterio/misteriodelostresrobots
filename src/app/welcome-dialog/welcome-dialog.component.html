Bienvenidos al Misterio de los tres robots

¡Hola! Este sencillo juego basado en diálogos es un ejemplo de uso de un modelo de aprendizaje automático dentro de una aplicación web. Puede consultar el código origial en https://github.com/google/mysteryofthreebots.

El modelo que utiliza se llama Universal Sentence Encoder Lite y está disponible en Tensorflow Hub.

Una herramienta útil para crear juegos basados en diálogos es Semantic Reactor, un complemento de Google Sheets que permite experimentar con modelos lingüísticos aprendidos por máquinas. Para obtener más información, visite la página de Google Cloud AI Workshop Page.

Para volver a ver esta pantalla, sólo tiene que recargar el juego en tu navegador. Y ahora... ¡a seguir con el juego!
